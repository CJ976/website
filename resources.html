Linux Command Line Essentials

  • ls -la — List all files with permissions
  • cd /path/to/dir — Change directory
  • cat file.txt — Display file contents
  • grep "pattern" file — Search for text pattern
  • chmod +x script.sh — Make file executable
  • ps aux | grep process — Find running process
  • netstat -tulpn — Show listening ports
  • find / -name "file.txt" — Search for files

Nmap Scanning Commands

  • nmap -sV -sC target.com — Service/version detection with default scripts
  • nmap -p- target.com — Scan all 65535 ports
  • nmap -A target.com — Aggressive scan (OS, services, scripts)
  • nmap -sU target.com — UDP scan
  • nmap -Pn target.com — Skip ping scan (useful for firewalls)
  • nmap --script vuln target.com — Run vulnerability scripts
  • nmap -oA output target.com — Output results to all formats
